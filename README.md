# game_of_life

A small but fun implementation of Conway's game of life in Rust using Kiss3d. The graphics processing doesn't seem to batch well, and I couldn't find a way to configure that in Kiss3d, so the larger the grid is, the slower it runs, haha!

The project can be run by cloning the repository, cd'ind into it, and running it using "cargo run". This was copied straight off of my system so there could be some trouble trying to run it without cleaning up some leftover files.
